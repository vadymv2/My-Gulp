 1. Установка GIT: https://git-scm.com/
   Во время установки обязательно установить опцию "Run Git from the Windows Command Prompt"

2. Установить Node.Js: https://nodejs.org/

3. Запускаем в консоли команду "npm -v" (без кавычек) и добиваемся показа технической информации о версии - это сигнал, что все ок, и Node + NPM уже установлены.

4. Простой командой:
   npm install -s bower
   Устанавливаем Bower

5. Немного кайфуем от самого процесса: одна команда - и файлы скачаны/установлены!

6. Ищем необходимые библиотеки:
   bower search jquery    или     bower s jquery
   или пользуемся поиском по сайту Bower: https://bower.io/search/

7. Установим пакет (библиотеку) командой install или i:
   bower install jquery

8. Для вывода списка всех установленных пакетов воспользуемся:
   bower list
   а также:
   bower list --path

9. Удалить пакет можно командой:
   bower uninstall jquery

10. Создаем файл-манифест bower.json (в обычной консоли Windows):
    bower init

11. Скачивание библиотеки с добавлением "в продакшен" и, как следствие, в bower.json:
    bower i jquery --save
    или:
    bower i jquery -S

12. Необходимо назначить новую "домашнюю папку" для Bower? Нет ничего проще:
    создаем файл .bowerrc
    Содержимое файла:
    {
        "directory" : "app/bower"
    }

13. Установим GULP:
    npm i gulp -d
